# Mason Examples

Real sites built with Mason. Each example shows the input (what was asked), the output (what was built), and the stack used.

---

## Contributing an Example

Built something with Mason? Open a PR adding a folder here:

```
examples/
└── your-site-type/
    ├── README.md        ← what you asked, what Mason built, screenshots
    ├── prompt.md        ← the exact conversation that kicked it off
    └── output/          ← the generated files (or a link to the live site)
```
